ClipChamp is fucking terrible.

It embodies everything that is wrong with the modern Microsoft Corporation.

It's bloated with tons of features no one asked for.
It chains you to a Microsoft account and nags you to upload everything to OneDrive.

Of course none of that matters if you can't reliably use it or keep it from crashing!

I hate ClipChamp so that's why I made RIPChamp.

RIPChamp is a back-to-basics clipping engine without the bullshit. A watcher agent monitors folders you choose for new video files. In a simple web interface you can see the queue of all the new videos ready to be clipped and proceed whenever you are ready. Inside the Trim page you will find a video preview and an intuitive interface for selecting the start and stop time of the new clip. Want to make it a local file? No problem. Want to upload to YouTube or Streamable? Sure thing. Post a link to the clip in Discord, we can handle that. Because that's what I wanted and I think some of you will want that too.

Discord Integration

RIPChamp posts finished clips to a Discord channel using a webhook. To create one:

  • Go to Server Settings → Integrations → Webhooks
  • Click New Webhook, pick the channel you want clips posted to, save your settings, then click Copy Webhook URL
  • Paste that URL below along with a name for the channel (just for your own reference -- this is what you'll pass to --discord-channel)
